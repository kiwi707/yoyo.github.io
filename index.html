Flclash 教程

安装 配置 运行 常见问题

1. 安装 Flclash 🔽

下载官方 GitHub 最新版本：

访问 GitHub 下载
  • 双击安装包，按默认选项完成安装
  • 安装完成后启动 Flclash

示意截图：

2. 配置 Clash 🔽

  • 打开 Flclash → 点击“配置文件管理”
  • 导入 Clash 配置文件（.yaml 或 .yml）
  • 选择配置文件 → 启用系统代理或全局模式

示意截图：

3. 运行 Flclash 🔽

  • 确认代理模式 → 点击“启动”
  • 查看日志，确认服务已启动
  • 尝试访问需要代理的网站，确认可用

示意截图：

4. 常见问题 🔽

  • 网站打不开：检查系统代理设置是否正确
  • 配置文件无法导入：确保文件格式为 .yaml 或 .yml
  • 启动失败：尝试以管理员权限运行 Flclash
