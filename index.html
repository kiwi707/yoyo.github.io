Flclash & Shadowrocket 教程

Flclash 安装 Flclash 配置 Flclash 运行 Shadowrocket 安装 Shadowrocket 配置 Shadowrocket 运行 常见问题

Flclash 安装 🔽

下载官方 GitHub 最新版本：

访问 GitHub 下载
  • 双击安装包，按默认选项完成安装
  • 安装完成后启动 Flclash

Flclash 配置 🔽

  • 打开 Flclash → 点击“配置文件管理”
  • 导入 Clash 配置文件（.yaml 或 .yml）
  • 选择配置文件 → 启用系统代理或全局模式

Flclash 运行 🔽

  • 确认代理模式 → 点击“启动”
  • 查看日志，确认服务已启动
  • 尝试访问需要代理的网站，确认可用

Shadowrocket 安装 🔽

在 App Store 下载 Shadowrocket：

  • 安装完成后打开应用
  • 允许必要权限

Shadowrocket 配置 🔽

  • 点击右上角“+” → 添加配置
  • 输入服务器信息或导入配置文件
  • 保存并启用配置

Shadowrocket 运行 🔽

  • 启用代理开关
  • 查看连接状态
  • 尝试访问需要代理的网站，确认可用

常见问题 🔽

Flclash FAQ

  • 网站打不开：检查系统代理设置是否正确
  • 配置文件无法导入：确保文件格式为 .yaml 或 .yml

Shadowrocket FAQ

  • 连接失败：检查服务器信息或网络是否通畅
  • 配置导入失败：确保配置文件格式正确
